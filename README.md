## 起因

由于我自己在拍视频时，需要一个无线麦克风来支持我的拍摄，所以开发了这个固件。

## 硬件

此固件支持我开源的无线麦克风的发射端硬件，硬件请参考hardware仓库。

## 使用

下载整个库，然后利用arduino IDE进行烧录即可。

## 赞助

如果项目对你有帮助，可以直接通过对相应视频点赞或者转发。
